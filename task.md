# Задание на разработку приложения "Домашняя библиотека"

## Описание
Необходимо разработать настольное приложение для управления персональной библиотекой. Приложение должно позволять хранить информацию о сборниках, авторах и рассказах, а также осуществлять навигацию между ними.

## Функциональные требования

### Основные сущности
1. **Сборники**
   - Название
   - Количество страниц

2. **Авторы**
   - Имя автора

3. **Рассказы**
   - Название
   - Автор (ссылка на автора)
   - Сборник (ссылка на сборник)
   - Количество страниц
   - Номер страницы в сборнике

### Функции приложения
1. Просмотр списков сборников, авторов и рассказов
2. Добавление новых сборников, авторов и рассказов
3. Отображение данных в виде таблиц с возможностью сортировки

### Технические требования
1. Язык программирования: Python
2. База данных: SQLite
3. Графический интерфейс: PyQt6
